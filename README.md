# SQL Fundamentals: Interactive Documentation

**[View Live Demo](https://tubolyroli.github.io/sql-fundamentals/)**

An interactive, searchable, and themeable documentation for SQL fundamentals, designed for students and data professionals.

## 🚀 Features

- **Searchable:** Full-text search across all sections and subsections.
- **Interactive TOC:** A dynamic sidebar that tracks your reading progress (Scrollspy).
- **Dark Mode Support:** Toggle between light and dark themes for better readability.
- **Copy-to-Clipboard:** One-click copying for all SQL code examples.
- **Checklist Integration:** Mark sections as completed to track your learning journey (persists in browser).
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## 📂 Project Structure

- `index.html`: The main interactive documentation (Primary user version).
- `main.tex`: LaTeX source for the documentation.

## 📖 How to Use

1. Clone the repository: `git clone https://github.com/tubolyroli/sql-fundamentals.git`
2. Open `index.html` in any modern web browser.
3. Use the sidebar to navigate or the search bar at the top to find specific SQL commands or concepts.

## 🛠️ Built With

- **HTML5/CSS3** (Modern CSS Grid & Sticky Headers)
- **Vanilla JavaScript** (Scrollspy, Search, and Persistence)
- **Highlight.js** (Syntax highlighting for SQL)
- **Inter & JetBrains Mono** (Typography)
